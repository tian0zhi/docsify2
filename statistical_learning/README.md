## **1.2监督学习与非监督学习**
### **1.2.1监督学习**
>1. 得到一个有限的训练数据集合（训练集）
>2. 确定模型的假设空间，也就是所有备选模型（模型结构）
>3. 确认模型学习的准则，即学习的策略（怎么学习）
>4. 实现求解最优模型的算法（如何求解）
>5. 通过学习方法选择最优模型
>6. 利用学习的最优模型对新数据进行预测或分析（预测）
